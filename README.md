# 猫猫手册

XDU 猫协公开信息展示站，涵盖校园流浪猫疫苗、绝育、领养、物资、SOP 和行动记录。

🏠 在线地址：https://thebear617.github.io/cat-knowledge/

## 页面板块

| 板块 | 内容 |
|------|------|
| 猫只档案 | 校园猫咪状态、疫苗进度、绝育信息、领养去向 |
| 物资管理 | 航空箱/猫包、猫粮分布追踪 |
| 近期计划 | 当前任务窗口和待确认事项 |
| 标准 SOP | 疫苗、绝育、狂犬处置等标准流程 |
| 猫猫编年史 | 按月折叠的行动时间线 |
| 猫协分工 | 义卖组、疫苗绝育组、赞助组、宣传财务组职责 |
